Setting up your environment

We are going to be developing a chatbot. We will be using the Microsoft Bot Framework.  This framework has implementations for Python, NodeJS and .Net Core . For this particular tutorial we will be using the C# language in .Net Core.

No past experience with C#. However, we familiarity with Procedural and Object Oriented programming is necessary to get the most of the session. I.e. defining variables, managing flow control with loops and conditional statements, recognising classes, and methods or functions.

Familiarity with Git and Visual Studio Code is also advantageous, as we will assume some understanding of navigating those tools and performing common tasks.

To begin, we need to setup environments. You will need to download and install the following:

     
  • Visual Studio Code - https://code.visualstudio.com/
  • Dotnet Core v. 2.20 https://dotnet.microsoft.com/download/dotnet-core/2.2 (download the latest SDK and runtime).
  • Git - https://git-scm.com/
  • Bot framework emulator - https://github.com/microsoft/BotFramework-Emulator/releases (click on the most recent version number and scroll down to find the download link).
